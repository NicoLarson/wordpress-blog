Comment fusionner des feuilles Excel dans une seule feuille ?

Date: 09 Fév 2022

#excel #vba #windows

Aujourd’hui, on m’a envoyé un fichier avec 200 onglets pour que je les regroupe dans une page Excel. J’ai essayé une méthode avec « consolider », mais ça n’a pas fonctionné, car les tableaux dans les onglets n’étaient pas configurés comme tableau. J’ai trouvé la méthode qui suit.

Les prérequis

  1. Les données doivent commencer à partir de « A1 ».

  2. Les données doivent posséder la même structure.

  3. Ce code peut uniquement combiner toutes les feuilles de calcul du classeur actif.

Les étapes

  1. Dans le classeur, appuyez sur Alt + F11 clés pour ouvrir l’application Microsoft Visual Basic.

  2. Cliquez sur insérer > Module, puis copiez-collez sous le code VBA ci-dessous dans le nouveau Module fenêtre.

Sub Combine()
    Dim i As Integer
    Dim xTCount As Variant
    Dim xWs As Worksheet
    On Error Resume Next
LInput:
    xTCount = Application.InputBox("The number of title rows", "", "1")
    If TypeName(xTCount) = "Boolean" Then Exit Sub
    If Not IsNumeric(xTCount) Then
        MsgBox "Only can enter number", , "Kutools for Excel"
        GoTo LInput
    End If
    Set xWs = ActiveWorkbook.Worksheets.Add(Sheets(1))
    xWs.Name = "Combined"
    Worksheets(2).Range("A1").EntireRow.Copy Destination:=xWs.Range("A1")
    For i = 2 To Worksheets.Count
        Worksheets(i).Range("A1").CurrentRegion.Offset(CInt(xTCount), 0).Copy _
               Destination:=xWs.Cells(xWs.UsedRange.Cells(xWs.UsedRange.Count).Row + 1, 1)
    Next
End Sub

  1. Pressez F5. Une boîte de dialogue apparaît pour entrer le nombre de lignes.

  2. Cliquez OK. Et toutes les feuilles du classeur actif seront fusionnées dans une nouvelle page appelée « Combiné ».
